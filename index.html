▸ Comment obtenir un Client ID ? (3 min) 1. Crée un projet Google Cloud
2. Va dans APIs & Services → Bibliothèque → active Google Drive API
3. Va dans APIs & Services → Identifiants → Créer des identifiants → ID client OAuth
4. Type : Application Web
5. Origines JS autorisées : ajoute l'URL de ton GitHub Pages
6. Copie le Client ID et colle-le ci-dessus
